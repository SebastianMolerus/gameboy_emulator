## How to build

1. visit : https://vcpkg.io/en/getting-started
2. select platform Windows/Linux. Do step 1&2.
3. cmake -B [build directory] -S . -DCMAKE_TOOLCHAIN_FILE=[path to vcpkg]/scripts/buildsystems/vcpkg.cmake
